# For Anjanaa ✨

A premium, minimal, and emotionally elegant website built as a digital letter.

## 🚀 Tech Stack
- **Next.js 15+** (App Router)
- **Tailwind CSS 4.0** (Modern CSS-first approach)
- **Framer Motion** (Smooth cinematic animations)
- **Lucide React** (Minimalist iconography)

## 📁 Structure
- `src/config/content.ts`: All text content is stored here for easy editing.
- `src/app/page.tsx`: The main experience with all sections.
- `public/images/centerpiece.png`: The main visual centerpiece.

## 🎨 Customizing
1. **Text**: Edit `src/config/content.ts`.
2. **Image**: Replace `public/images/centerpiece.png` with your favorite photo of/with Anjanaa.
3. **Music**: In `src/app/page.tsx`, you can replace the `new Audio()` URL with a link to her favorite soft instrumental song or a local path like `/audio/music.mp3`.

## 🛠️ How to Run
1. Install dependencies: `npm install`
2. Start dev server: `npm run dev`
3. Open `localhost:3000` in your browser.
